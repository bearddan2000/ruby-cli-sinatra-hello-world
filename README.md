# ruby-cli-sinatra-hello-world

## Description
A POC project for ruby sinatra.

## Tech stack
- bundle
- ruby
  - sinatra
  - puma

## Docker stack
- ruby:latest

## To run
`sudo ./install.sh -u`
- [Default](http://localhost)
- [Parameterized](localhost/hello/adam)

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
